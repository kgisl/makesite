Transaction Processing System

Mini project for CS3251 Lab

  1. Review the code at https://replit.com/@ashokb/Unit5Programs#trans.c
  2. Compile, and run the executable code
  3. Do your own testing of the program
  4. What happens when you add a new account detail? And then generate accounts.txt file again. Inspect the contents. Is the contents as you expected them to be?
  5. If there are some errors, fix the program logic.
  6. Add some new functionality. How about listing the accounts information?
  7. Can you make the program more efficient?
